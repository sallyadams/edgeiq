# 🚀 EdgeIQ — Smart Market Intelligence Platform

> See what the smart money is doing — in real time.

EdgeIQ is a modern market intelligence platform designed to surface institutional flows, insider activity, and high-signal trading insights. Built for traders, analysts, and data-driven decision-makers.

---

## ✨ Features

* 📊 Real-time market intelligence
* 📈 Smart money tracking & signal detection
* ⚡ Fast, modern UI powered by Vite + React
* 🔐 Authentication via OpenID (Replit integration)
* 🧠 Data-driven insights for better trading decisions
* 🎯 Clean, minimal UX focused on clarity

---

## 🖥️ Tech Stack

* **Frontend:** React + TypeScript + Vite
* **Package Manager:** pnpm
* **Auth:** OpenID Connect (Replit)
* **Build Tools:** esbuild / lightningcss
* **Monorepo:** pnpm workspaces

---

```
edgeiq/
```

## ⚙️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/edgeiq.git
cd edgeiq
```

### 2. Install dependencies

```
pnpm install
```

### 3. Run the development server

```
pnpm dev
```

### 4. Open in browser

```
http://localhost:3000
```
